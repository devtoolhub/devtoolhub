## What is DevToolHub?

[DevToolHub](https://devtoolhub.net) is a growing collection of free browser-based developer tools. Every tool runs 100% client-side — your data never touches a server.

The site is designed to be fast, ad-free (for now), and privacy-first. Built with Astro and React.

## Privacy

Every tool on DevToolHub runs **entirely in your browser**. Your data is never sent to any server. There are no analytics trackers that collect personal information. No account or signup required.

## Roadmap

DevToolHub is actively developed. The goal is **300+ tools** by the end of 2026, covering:

- **Text processing** — counters, cleaners, case converters, find & replace
- **More encoders** — ROT13, Unicode escape, Hex, Binary, Punycode, Morse code
- **Code formatters** — CSS, HTML, XML, JavaScript, TypeScript, YAML, GraphQL
- **Minifiers** — CSS, HTML, JavaScript
- **Data converters** — CSV, XML, TOML, Markdown tables, SQL
- **JSON to code** — TypeScript interfaces, Go structs, Python dataclasses, and more
- **Crypto tools** — SHA-512, HMAC, Bcrypt, AES encrypt/decrypt, RSA key generation
- **CSS generators** — gradients, shadows, flexbox, grid, filters, transforms
- **Network tools** — HTTP status reference, MIME lookup, User Agent parser, cURL converter
- **DevOps** — Docker Compose validator, K8s YAML checker, Nginx formatter, systemd generator
- **Validators** — XML, YAML, TOML, HTML, CSS, JSON Schema, email, URL, IP
- **Cheat sheets** — HTTP status codes, ASCII table, SQL keywords, Git commands, and more
